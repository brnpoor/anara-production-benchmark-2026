Why this exists

Production benchmarks reveal failures synthetic tests miss.

The ANARA benchmark asked AI coding systems to audit a live production deployment, not a toy repository. The point was not to crown a model. The point was to test whether AI systems can reason about real infrastructure, live logs, security exposure, provider routing, deployment drift, and voice-agent failure modes.

Security

Authentication bypass

Multiple evaluations surfaced public LLM endpoint exposure and caller identity spoofability as production blockers.

Operations

Deployment drift

The benchmark separated codebase claims from what was actually serving in Cloud Run.

Reliability

Provider degradation

The audit distinguished working providers from misconfigured or invalid credentials.

Acceptance

Voice validation failure

Wingman phone testing found task-oriented conversation failures that would not appear in a static code review.
